# Brand Guidelines: Elegant Professional

**Version**: 1.0.0
**Status**: APPROVED

---

## 1. Brand Philosophy

**Tagline**: "Connecting Global Hair Innovation to Indonesia's Salon Professionals"
**Voice**: Professional, Knowledgeable, Elegant, Reliable.
**Aesthetic**: "Shadow & Depth" (Not Flat).

## 2. Typography

**Primary Font**: Montserrat (Variable)

- **Hero**: Bold (700), Tight Tracking (-0.04em)
- **Headings**: SemiBold (600), Tight Tracking (-0.03em)
- **Body**: Regular (400), Relaxed Leading (1.75)

## 3. Color Palette (Monochrome)

| Token | Hex | Usage |
|---|---|---|
| `--foreground` | `#09090b` | Primary Text |
| `--muted` | `#71717a` | Secondary Text |
| `--background` | `#ffffff` | Page Base |
| `--panel` | `#fafafa` | Card Component Base |

## 4. UI Standards (Do's & Don'ts)

### ✅ Do

- Use **Shadows** (`--shadow-elegant`) to create lifting depth.
- Use **Staggered Reveals** for all text blocks on scroll.
- Use **Rounded Corners** (24px/32px) for cards and images.
- Use **Glassmorphism** for sticky headers.

### ❌ Don't

- Do NOT use flat colors without depth (Material Design v1 style).
- Do NOT use generic system fonts (Times, Arial).
- Do NOT use "pop" colors (Red/Blue/Green) except for status indicators.
- Do NOT use instant transitions; always use `--transition-elegant`.
